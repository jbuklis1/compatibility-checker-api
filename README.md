# AI-Powered Cross-Platform Compatibility Checker API

FastAPI + Uvicorn API that runs rule-based cross-platform checks, plus Together.ai for fix suggestions and test-case generation.

**Project layout:** The `app` folder contains the API; `cross_platform_checker` (sibling to `app`) contains the rule-based checker.

## Setup

1. Create a virtualenv and install deps:

   ```bash
   cd compatibility-checker-api
   python -m venv .venv
   source .venv/bin/activate   # or .venv\Scripts\activate on Windows
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and set `TOGETHER_API_KEY`. Get a key at [Together AI](https://api.together.xyz/settings/api-keys).

3. Run from inside `compatibility-checker-api`:

   ```bash
   cd compatibility-checker-api
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

   Or use the setup script to create a venv, install deps, and start the app:

   ```bash
   cd compatibility-checker-api
   ./setup_and_run.sh
   ```

   On Windows:

   ```batch
   cd compatibility-checker-api
   setup_and_run.bat
   ```
